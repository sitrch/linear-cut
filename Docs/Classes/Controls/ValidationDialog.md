# Описание класса ValidationDialog

## Уровень: Local (Level 2)
**Пространство имен:** `LinearCutWpf.Controls`  
**Сборка:** `LinearCutWpf`  
**Тип:** `Window`  

## Назначение
Класс `ValidationDialog` представляет собой диалоговое окно, которое отображается в случае обнаружения ошибок валидации при подготовке данных к раскрою. Окно выводит таблицу с описанием ошибок и позволяет пользователю либо прервать процесс, либо проигнорировать предупреждения и продолжить выполнение операции.

## Зависимости (DI и связи)
- **DataTable:** Принимает `DataTable` с данными об ошибках (обычно формируется в `GroupingControl`).

## Основные свойства и методы
- **Свойство `Ignored`** (`bool`): Указывает, нажал ли пользователь кнопку "Игнорировать", что позволяет продолжить выполнение процесса несмотря на наличие ошибок.
- **Конструктор `ValidationDialog(DataTable errorsTable)`**: Инициализирует окно и устанавливает источник данных для `DataGrid` (список ошибок).
- **`BtnIgnore_Click`**: Обработчик кнопки игнорирования ошибок. Устанавливает `Ignored = true`, закрывает окно с результатом `true`.
- **`BtnCancel_Click`**: Обработчик кнопки отмены. Устанавливает `Ignored = false`, закрывает окно с результатом `false`.
