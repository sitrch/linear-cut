# Models

**Namespace:** `LinearCutWpf.Models`
**Path:** `Models/Models.cs`

## Описание
В файле `Models.cs` собраны основные структуры данных и модели (POCO), используемые в приложении для представления настроек, деталей, хлыстов и состояния UI. Многие классы реализуют интерфейс `INotifyPropertyChanged` для обеспечения двусторонней привязки данных (Data Binding) в WPF.

## Основные классы

### CutSettings
Глобальные настройки раскроя.
- **Свойства:** `TrimStart` (торцевание в начале), `TrimEnd` (торцевание в конце), `CutWidth` (толщина реза), `ItemStocks` (словарь доступных длин хлыстов по артикулам).

### PresetModel
Модель пресета настроек.
- **Свойства:** `Name`, `TrimStart`, `TrimEnd`, `CutWidth`.

### CutBar
Модель результата распила одного хлыста (базовая).
- **Свойства:** `StockLength` (длина исходного хлыста), `Parts` (строковое представление расположенных деталей), `Remainder` (остаток).

### CutBarDetailed
Детализированная модель распила хлыста. Используется на финальной стадии отображения результатов.
- **Свойства:** `StockLength`, `Remainder`, `Parts` (список объектов `PartItem`).

### PartItem
Модель конкретной детали на хлысте.
- **Свойства:** `Article`, `Length`, `Count`, `OriginalRowIndex` (индекс оригинальной строки данных для отслеживания).

### StockLengthModel
Модель доступной длины хлыста в настройках данных.
- **Свойства:** `Length`, `IsEnabled` (флаг активности для алгоритма).

### ManualCutRow
Сложная модель строки ручного ввода деталей для определенного хлыста.
- **Назначение:** Представляет одну строку в UI ручного раскроя. Содержит логику валидации (`HasLengthError`, `HasCountError` и др.) и динамический расчет доступного места (`AvailableSpaceForSize1` и т.д.).
- **Особенности:** Реализует `INotifyPropertyChanged`. Хранит списки `PartSizeAvailability` для выпадающих списков.

### ArticleSettings
Индивидуальные настройки для конкретного артикула, которые переопределяют глобальные.
- **Свойства:** `ArticleName`, `ArticleDescription`, `BarLength` (длина хлыста по умолчанию для этого артикула), `Preset`, `ManualCuts` (коллекция строк ручного раскроя).
- **Методы:** `HasCustomSettings` - проверяет, отличается ли настройка артикула от глобальных по умолчанию.

### ArticleGroupingRow
Строка группировки деталей по артикулу для UI (вкладка "Группировка").
- **Свойства:** `ArticleName`, `ArticleDescription`, `TotalCount`, `TotalLength`, `SelectedBarLength`, `SelectedPreset`, `SelectedVisibleHeight`, `IsDefaultValue`, `IsManuallyChanged`.
- **Особенности:** Реализует `INotifyPropertyChanged`. Позволяет пользователю задавать индивидуальные настройки прямо из таблицы группировки. Используется для отслеживания состояния высоты профиля (значение по умолчанию, ручное изменение).

### PartSizeAvailability
Модель для отображения доступности размера детали (подходит ли он по длине в оставшееся место хлыста) в выпадающих списках ручного ввода.

## Зависимости
- `System.ComponentModel` (`INotifyPropertyChanged`)
- Нет внешних зависимостей. Чистые структуры данных.
