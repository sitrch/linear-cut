# Описание класса BoolToColorConverter

## Уровень: Local (Level 2)
**Пространство имен:** `LinearCutWpf.Converters`  
**Сборка:** `LinearCutWpf`  
**Тип:** `IValueConverter`  

## Назначение
Класс `BoolToColorConverter` реализует интерфейс `IValueConverter` и предназначен для использования в привязках данных WPF. Он конвертирует логическое значение (`bool`) в объект `Brush` для изменения цвета элементов интерфейса в зависимости от состояния (например, выделение выбранных строк в `DataGrid`).

## Зависимости (DI и связи)
- **WPF DataBinding:** Используется в XAML файлах (например, `ArticleGroupingControl.xaml`) для привязки свойства `Background` к логическим свойствам моделей.

## Основные элементы и методы
- **`Convert`**: Принимает `value` типа `bool`. Если значение `true`, возвращает светло-зеленую кисть (`Brushes.LightGreen`), в противном случае возвращает белую кисть (`Brushes.White`).
- **`ConvertBack`**: Не реализован (`throw new NotImplementedException()`), так как обратная конвертация из цвета в логическое значение в данном приложении не требуется.
