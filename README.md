# Линейный раскрой профиля

Приложение для оптимизации линейного раскроя профилей из хлыстов с использованием данных из Excel файла.

## Возможности

- Загрузка данных заготовок из Excel (артикул, цвет, длина, количество)
- Поддержка нескольких длин хлыстов для одного профиля
- Оптимизация раскроя по минимальному количеству хлыстов
- Учет параметров раскроя (обрезка в начале/конце, ширина реза)
- Сохранение результатов раскроя и остатков
- Управление остатками с ограничением по минимальной длине

## Структура Excel файла

### Лист "Заготовки"
| Артикул | Цвет | Длина | Количество |
|---------|------|-------|-----------|
| PROF-01 | Белый | 450 | 10 |
| PROF-01 | Белый | 650 | 5 |

### Лист "Профили"
| Артикул | Длина хлыста |
|---------|--------------|
| PROF-01 | 6000 |
| PROF-01 | 3000 |

### Лист "Параметры"
| Артикул | Обрезать в начале | Обрезать в конце | Ширина реза |
|---------|------------------|-----------------|------------|
| PROF-01 | 50 | 50 | 4 |

## Установка и запуск

1. Откройте решение в Visual Studio
2. Восстановите NuGet пакеты (ExcelDataReader, ExcelDataReader.DataSet)
3. Скомпилируйте проект
4. Запустите приложение

## Использование

1. Укажите путь к Excel файлу
2. Нажмите "Загрузить данные"
3. Нажмите "Оптимизировать" для запуска алгоритма раскроя
4. Просмотрите результаты на вкладке "Результаты раскроя" и "Остатки"
5. Нажмите "Сохранить результаты" для экспорта в CSV файлы

## Результаты

- `results.csv` - результаты раскроя по каждому хлысту
- `remnants.csv` - остатки материала (длина >= минимального порога)

## Требования

- .NET Framework 4.7.2
- Windows Forms
- Excel 2010 или выше (для источника данных)
